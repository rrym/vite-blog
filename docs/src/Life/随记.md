# 没工作也要好好爱自己

最近面试机会比较少，上周4个 这周还没
不过生活还是要过的，也要按时睡觉、起床、吃早餐、学习、继续努力投简历面试。

## 作息时间

### 早上

 早上大概7点左右醒来，玩一下去洗簌，然后准备一下吃点早餐，吃完开始学习 10点-10点半开始投简历

 ### 中午

 中午出门吃饭（主要是要溜达一下 呆在家里比较闷）然后回来午休（随便玩点什么手机、睡觉什么的）12点到1点半  下午继续学习（效率不高就是）

 ### 晚上

 傍晚6点准时出门吃饭，饭后散步半小时。晚上回来刷手机、论团、感悟人生 洗澡睡觉，尽量在12点前睡。（如果可以在提前一点睡也是可以的）
